Análisis de sensibilidad a la rugosidad — SFINCS¶

Caso de estudio: Río Besaya — Corrales de Buelna
Modelo: SFINCS 2D GPU (Deltares), resolución 5 m, T=500 años

Se analizan 1.000 simulaciones con distintas combinaciones de Manning.

Bugs corregidos respecto al análisis original:

  • Los TIFFs se cargan en orden numérico (load_flood_ensemble) — no léxico.
  • La estadística espacial usa mean(dim=['x','y']) — no secuencial.
  • El área inundada usa factor 1e-6 para m² → km² — no 10e-6.
  • El ensemble de Manning está alineado con el de resultados por simulation.

Rutas — ajustar según el entorno¶

1 · Cargar ensemble de resultados SFINCS¶

load_flood_ensemble extrae el número de simulación de cada nombre de archivo y ordena numéricamente, evitando el bug del orden léxico de glob.

Ensemble SFINCS: ('simulation', 'y', 'x') → shape (1000, 1201, 679)
Simulaciones: [  0   1   2 998 999]  (primeras/últimas)

2 · Cargar ensemble de rugosidad de Manning¶

build_manning_ensemble itera sobre los mismos números de simulación que el ensemble de resultados, garantizando la alineación.

Ensemble Manning: ('simulation', 'y', 'x') → shape (1000, 600, 339)
