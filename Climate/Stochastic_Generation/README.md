# Stochastic Generation — Generación Estocástica de Clima

Módulo para la generación de series temporales sintéticas de precipitación estadísticamente equivalentes a las observaciones.

## Código de referencia

- **NEOPRENE** (IHCantabria): [github.com/IHCantabria/NEOPRENE](https://github.com/IHCantabria/NEOPRENE)
- **CoSMoS_py** (navass11): [github.com/navass11/CoSMoS_py](https://github.com/navass11/CoSMoS_py)

El código desarrollado en estos repositorios es el punto de partida para los módulos `Point/` y `Spatial/` de esta librería.

## Submódulos

### Point
Generadores de clima a escala de punto (una única ubicación).

### Spatial
Generadores de campos espaciales de precipitación multi-sitio.
